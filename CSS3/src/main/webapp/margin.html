margin 요소의 외부 여백

테두리(border)를 기준으로 바깥쪽으로 사방에 적용 됨.

1. margin의 방향별 지정 속성

  • margin-top: 50px; 위에 50px만큼 적용
  • margin-right: 30%; 우측에 30%만큼 적용
  • margin-bottom: 2em; 아래에 2em만큼 적용
  • margin-left: 5%; 좌측에 5%만큼 적용

2. margin의 값 갯수별 지정법

  • margin:30px; 양사방에 같은 값을 동시 적용
  • margin:50px 70px; 상하 좌우 순서로 적용
    항상 정중앙에 위치하는 요소
    margin:n auto;
    (margin-left: auto;
    margin-left: auto;)
  • margin:20px 100px 0; 상 좌우 하 순서로 적용
  • margin:0 20% 100px 3%; 상 우 하 좌 순서로 적용

3. inline(a)의 margin 적용
