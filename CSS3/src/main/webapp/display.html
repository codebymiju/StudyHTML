display

요소가 브라우저 화면에 나타나는 방식 지정

  • display:inline;
  • display:inline-block;
  • display:none;
  • display

display의 값들

  • display:inline; block인 요소를 inline으로 바꿈
  • display:block; inline인 요소를 block으로 바꿈
  • display:inline-block; inline과 block을 합친 성격의 요소로 바꿈
  • display:none; 요소가 사라짐

1. display:inline; block인 요소를 inline으로

block인 p

block인 div
block인 aside

2. display:block; inline인 요소를 block으로

inline인 a inline인 b inline인 span

3. display:inline-block; inline과 block을 합친 성격의 요소로 바꿈

block인 p

block인 div
block인 aside
inline인 a inline인 b inline인 span

4. display:none; (영역을 남기지 않고) 요소가 사라짐

block인 p

block인 div
block인 aside
inline인 a inline인 b inline인 span

5. visibility:hidden; (영역은 남기고) 요소가 사라짐

block인 p

block인 div
block인 aside
inline인 a inline인 b inline인 span
