css basic

선택자 { 속성 : 속성값; }

CSS 선택자(Selector) 대표 3가지

  • tag 선택자 - tag 그대로를 선택자로 활용. 강제, 자동 적용.
  • id 선택자 - tag에 id를 추가하고 #id형태로 활용.
    id값이 중복적용 불가하므로 해당 스타일 또한 중복적용 불가
    주로 구조적인 목적에 쓰임
  • class 선택자 - tag에 class를 추가하고 .class형태로 활용.
    class값은 중복적용 가능하므로 주로 디자인적인 목적에 쓰임.

css적용법 3가지

  • External CSS - 외부파일 방식
    스타일소스를 별도의 파일로 독립해 작성후 웹문서와 연결하여 적용.
    <웹문서의 head영역에> <link href = "*.css" rel = "stylesheet">
    통일성 있는 구성 가능, 유지보수 유리. (장점 많음)
  • Internal CSS - 내장 방식
    스타일소스를 웹문서의 head영역에 작성.
    <style type = "text/css"> 선택자 {속성:속성값;} </ style >
    해당 웹문서에만 적용 됨.
  • Inline CSS - 인라인 방식
    스타일소스를 해당 태그 바로 뒤에 작성.
    <tag style = "속성:속성값;">
    CSS의 장점이 없으나 세가지 방식 중 CSS적용 우선 순위가 가장 높고,
    body 부분만 수정 가능한 상황에서 활용.
