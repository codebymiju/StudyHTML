padding 요소의 내부 여백

테두리(border)를 기준으로 안쪽으로 사방에 적용 됨.

1. padding의 방향별 지정 속성

  • padding-top: 1.5em; 위에 1.5em만큼 적용
  • padding-right: 20%; 우측에 20%만큼 적용
  • padding-bottom: 1.2em; 아래에 1.2em만큼 적용
  • padding-left: 50px; 좌측에 50px만큼 적용

2. padding의 값 갯수별 지정법

  • padding:n ; 양사방에 같은 값을 동시 적용
  • padding:n n; 상하 좌우 순서로 적용
  • padding:n n n; 상 좌우 하 순서로 적용
  • padding:n n n n; 상 우 하 좌 순서로 적용
